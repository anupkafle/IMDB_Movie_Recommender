# IMDB_Movie_Recommender
Movie Recommendation system using Clustering

## Srapping Data
All the data were scrapped from the [IMDB](https://www.imdb.com/) website

  * Install requirements.txt
      > pip install -r requirements.txt
  * Run manin.py
      > python main.py
